# MyStockControl



**MyStockControl** es una aplicación de escritorio intuitiva y eficiente, diseñada para facilitar la administración de inventarios en comercios pequeños, kioscos y emprendimientos. El sistema centraliza el control de productos, optimizando la organización y eliminando los errores comunes asociados a la gestión manual.

---

## 🚀 Características Principales

* **Gestión Integral de Inventario**: Registro completo de productos, incluyendo nombre, categoría, precio y cantidad inicial.
* **Control de Ventas Automatizado**: Registro de transacciones con actualización en tiempo real de las existencias disponibles.
* **Detección de Stock Crítico**: Identificación automática de productos agotados para facilitar la gestión de reposición.
* **Visualización Dinámica**: Interfaz organizada en tablas para una consulta rápida y clara del estado del negocio.
* **Reportes en PDF**: Funcionalidad integrada para exportar el inventario actual a formato PDF.

## 🛠️ Tecnologías Utilizadas

El sistema fue construido bajo los principios de la Programación Orientada a Objetos utilizando:

* **Lenguaje**: Python.
* **Interfaz Gráfica**: wxPython.
* **Base de Datos**: SQLite.

## 📋 Funcionamiento

La aplicación ha sido diseñada para una experiencia de usuario fluida:

1. **Pantalla de Carga (Splash Screen)**: Un inicio profesional con una duración controlada de 3 segundos.
2. **Navegación Intuitiva**: Panel lateral fijo que permite alternar entre funciones sin necesidad de ventanas emergentes adicionales.
3. **Validaciones de Seguridad**: El sistema cuenta con alertas de error ante campos incompletos o intentos de venta con stock insuficiente.
